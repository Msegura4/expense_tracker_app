# Expense Tracker App

Application de suivi de dépenses avec extraction automatique depuis des reçus.

## Branches

| Branche | Description |
|---|---|
| `feature/extraction-backend` | API backend pour l'extraction de données |
| `feature/google-sheets` | Intégration Google Sheets |
| `feature/frontend-vibe` | Interface utilisateur |
| `feature/edition-manuelle` | Édition manuelle des dépenses |
| `feature/upload-image-sheet` | Upload et traitement d'images de reçus |
